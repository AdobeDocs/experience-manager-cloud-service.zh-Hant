# 在互動式通訊編輯器中複製並貼上

>[!NOTE]
>
> 互動式通訊功能在早期採用者程式中提供。 使用您的工作郵件地址寄送電子郵件至 `aem-forms-ea@adobe.com` 請求存取權限。

>[!IMPORTANT]
>
> **文件內容可能隨時變更**：此提示庫目前正在針對產品進行測試，可能會進行更新和修訂。隨著表單體驗建立工具在早期採用者方案期間持續發展，相關的提示、範例和最佳做法可能會有所變更。

互動式通訊(IC)編輯器中的「複製並貼上」功能可讓作者複製現有的互動式通訊，並在不同的資料夾或位置重複使用它。 此功能可啟用多種積體電路變化，而不需修改原始版本，以協助團隊有效率地工作。

透過建立副本，作者可以安全地針對不同使用案例、客戶或業務案例進行變更，同時保留來源互動式通訊。

## 要複製的內容

當您複製互動式通訊時，以下元素會複製到新的IC中：

- 版面配置與結構

- 元件和片段

- 資料繫結和規則

- 樣式和品牌設定

- 然後，您可以在複製的IC中單獨修改這些元素中的任何一個。

## 複製互動式通訊

![尋找IC檔案](/help/forms/interactive-communication/assets/copy-in-ic.png)

請依照下列步驟複製互動式通訊：

- 在AEM介面中導覽至Forms &amp; Documents 。

- 找到您要複製的互動式通訊。

- 選取互動通訊。

- 從工具列或快顯選單中選擇「複製」。

- 導覽至目標資料夾或位置。

- 選取「貼上」以建立互動式通訊的復本。

- 新的IC會以與原始積體電路相同的組態建立。

## 編輯並重複使用複製的互動式通訊

![尋找IC檔案](/help/forms/interactive-communication/assets/paste-in-ic.png)

貼上互動式通訊後：

- 在互動式通訊編輯器中開啟複製的IC。

- 視需要更新內容、規則、資料對應或版面配置。

- 準備就緒後儲存並發佈互動通訊。

- 對複製的IC所做的變更不會影響原始的互動式通訊。

## 最佳做法

- 保持原始IC為基準或主版本。

- 請清楚地重新命名複製的IC，以反映其目的或變化。

- 結合使用複製並貼上版本設定和註解以提升追蹤能力。

- 複製後請檢閱資料繫結和輸出通道，以確保正確性。

互動式通訊中的「複製並貼上」功能可讓作者複製現有的IC並獨立修改它們，進而簡化重複使用和自訂作業。 它可讓您更快速地開發、更安全的實驗，以及一致的通訊傳遞，而不會冒變更原始互動式通訊的風險。
