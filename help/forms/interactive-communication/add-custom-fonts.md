# 新增自訂字型至互動式通訊編輯器

>[!NOTE]
>
> 互動式通訊功能在早期採用者程式中提供。 使用您的工作郵件地址寄送電子郵件至 `aem-forms-ea@adobe.com` 請求存取權限。

自訂字型功能可讓您使用自訂或組織核准的字型，以強化互動式通訊的外觀與感覺。 有了這項功能，您可以將字型封裝並嵌入到AEM Forms as a Cloud Service環境中，確保跨裝置和平台呈現一致的PDF。

## 主要優點

- 使用組織核准的字型維持品牌一致性。

- 即使字型未安裝在本機，請確定裝置間的PDF呈現方式正確無誤。

- 提供專業且統一的通訊體驗。

## 新增自訂字型至互動式通訊

>[!NOTE]
>
> 如需詳細的設定步驟，使用者可以遵循指南[使用自訂字型](/help/forms/use-custom-fonts.md)
>完成步驟並執行管道後，自訂字型可直接在互動式通訊編輯器中使用。
