# 簡介 {#introduction}

Adobe Experience Manager Forms as a Cloud Service 為企業提供了一個雲端原生的平台即服務 (PaaS) 解決方案，用於建立、管理、發佈和更新複雜的數位表單，同時將提交的資料與後端流程、業務規則相整合，並將資料儲存在外部資料存放區中。該服務始終是最新的，始終可用，並且始終都在學習。
