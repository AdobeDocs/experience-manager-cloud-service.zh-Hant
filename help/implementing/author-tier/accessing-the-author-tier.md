# 存取作者階層 {#accessing-the-author-tier}

AEMas a Cloud Service環境的作者層級可透過標準且不可自訂的URL來存取。

URL格式如下：

* `https://author-p[Program ID]-e[EnvironmentID].adobeaemcloud.com/`

不支援AEM製作層的自訂網域名稱。
