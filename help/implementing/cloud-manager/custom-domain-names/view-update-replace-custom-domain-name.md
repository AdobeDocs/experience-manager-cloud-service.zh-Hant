# 查看、更新和替換自定義域名 {#view-update-replace-cdn}

使用「查看」和「更新」菜單，只需查看一個或多個自定義域名的詳細資訊。
或者，可以更改用於自定義域名的SSL證書。

>[!NOTE]
>SSL證書必須有效並包含自定義域名。
