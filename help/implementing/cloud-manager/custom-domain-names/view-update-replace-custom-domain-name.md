# 查看和更新和替換自定義域名{#view-update-replace-cdn}

使用「查看和更新」菜單，只需查看一個或多個自定義域名的詳細資訊。
或者，您也可以變更SSL憑證以用於自訂網域名稱。

>[!NOTE]
>SSL憑證必須有效且包含自訂網域名稱。
