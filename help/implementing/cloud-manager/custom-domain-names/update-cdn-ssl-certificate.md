# 更新自訂網域名稱的SSL憑證{#update-ssl-certificate-cdn}

您可以從UI中的兩個不同位置執行更新動作：

1. 從「環境」畫面導覽至「**網域設定**」頁面。
1. 或者，從&#x200B;**Environments**&#x200B;螢幕導航到特定的「環境詳細資訊」頁，然後導航到「域名」表。
   >[!NOTE]
   >標識列出要查看或更新的自定義域名的行。
1. 選擇&#x200B;**...**&#x200B;功能表
1. 選擇&#x200B;**View &amp; Update**&#x200B;選項。
1. 視需要變更SSL憑證並確認您的提交。
