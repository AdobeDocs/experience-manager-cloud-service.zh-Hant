# 更新自訂網域名稱的SSL憑證{#update-ssl-certificate-cdn}

更新動作可從UI的兩個不同位置執行：

1. 導覽至「環境」畫面中的「網域設定」**頁面。**
1. 或者，從&#x200B;**Environments**&#x200B;螢幕導航到特定的「環境詳細資訊」頁，並導航到「域名」表。
   >[!NOTE]
   >標識要查看或更新的自定義域名列在的行。
1. 選擇&#x200B;**...**&#x200B;選單，位於列的最右端
1. 選擇&#x200B;**查看和更新**&#x200B;選項。
1. 視需要變更SSL憑證並確認您的提交。
