# 取得自訂網域名稱 {#getting-cdn}

您可以向域名註冊機構、管理和銷售域名的公司或組織購買域名。 域名註冊器管理域名系統(DNS)伺服器上的域名。

>[!NOTE]
>Cloud Manager不是域名註冊機構，不提供DNS服務。

DNS可讓我們將機器可讀的電腦位址（例如易記的位址）轉換為機器可讀的位址，將自訂的品牌化網域附加 `184.33.123.64` 至網站 `www.wknd.com`。
