# 獲取自定義域名 {#getting-cdn}

您可以從域名註冊機構、管理和銷售域名的公司或組織購買域名。 域名註冊器管理域名系統(DNS)伺服器上的域名。

>[!NOTE]
>雲管理器不是域名註冊器，不提供DNS服務。

DNS允許我們通過翻譯機器可讀的電腦地址(如 `184.33.123.64` 到難忘的地址，比如 `www.wknd.com`。
