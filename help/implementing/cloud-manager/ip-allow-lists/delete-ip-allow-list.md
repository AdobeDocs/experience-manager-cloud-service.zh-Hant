# 刪除IP允許清單 {#delete-allow-list}

按照以下步驟刪除IP允許清單：

1. 導航到 **域設定** 的 **環境** 的上界。
1. 標識要刪除的自定義域名所在的行。
1. 選擇 **...** 的子菜單。
1. 選擇 **刪除** 的雙曲餘切值。
1. 確認提交。

>[!NOTE]
>刪除IP允許清單將自動取消應用該清單中的所有服務並從表中將其刪除。
