# 刪除IP允許清單{#delete-allow-list}

請依照下列步驟刪除IP允許清單：

1. 從「環境」畫面導覽至「網域設定」頁面。
1. 標識要刪除的自定義域名所在的行。
1. 選擇&#x200B;**...**&#x200B;選單。
1. 選擇「刪除」選項。
1. 確認您的提交。

>[!NOTE]
>刪除IP允許清單將自動取消應用該清單中的所有服務，並從表中刪除該服務。
