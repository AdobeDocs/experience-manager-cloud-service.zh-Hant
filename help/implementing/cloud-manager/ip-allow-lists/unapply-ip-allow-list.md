# 取消應用IP允許清單{#apply-allow-list}

取消套用IP允許清單是環境中，允許清單定義中包含的所有IP範圍會與製作/或發佈者服務取消關聯的程式。

>[!NOTE]
>必須登錄業務所有者或部署管理員角色中的用戶，才能取消應用IP允許清單。

請依照下列步驟取消套用IP允許清單：

1. 從&#x200B;**Environments**&#x200B;螢幕導覽至特定的「環境詳細資訊」頁面，並導覽至「IP允許清單」表格。
1. 識別您要取消套用的IP允許清單規則所列的列。
1. 選擇&#x200B;**...**&#x200B;功能表。
1. 選擇&#x200B;**取消應用**&#x200B;選項並確認提交。
