# 應用和取消應用IP允許清單 {#apply-allow-list}

應用IP允許清單時，清單定義中包含的所有IP範圍都與環境中的作者或發佈服務相關聯。 取消應用清單是此進程的反向。

## 應用IP允許清單 {#applying}

中的用戶 **業務所有者** 或 **部署管理器** role可以按照以下步驟應用IP允許清單。

1. 登錄到Cloud Manager(位於 [my.cloudmanager.adobe.com](https://my.cloudmanager.adobe.com/) 並選擇相應的組織和程式。
1. 導航到 **環境** 螢幕 **概述** 的子菜單。
1. 導航至上的特定環境詳細資訊頁面 **環境** 螢幕並導航至 **IP允許清單** 的子菜單。
1. 使用表頂部的輸入欄位選擇IP允許清單以及要應用它的作者或發佈服務。
   * IP允許清單必須存在於雲管理器中才能應用。
1. 按一下 **應用** 並確認您的提交。

## 取消應用允許清單 {#un-applying}

中的用戶 **業務所有者** 或 **部署管理器** role可以按照以下步驟取消應用IP允許清單。

1. 登錄到Cloud Manager(位於 [my.cloudmanager.adobe.com](https://my.cloudmanager.adobe.com/) 並選擇相應的組織和程式。
1. 導航到 **環境** 螢幕 **概述** 的子菜單。
1. 導航至上的特定環境詳細資訊頁面 **環境** 螢幕並導航至 **IP允許清單** 的子菜單。
1. 確定要取消應用的IP允許清單的行。
1. 選擇行最右端的省略號按鈕。
1. 選擇 **取消應用** 確認您的提交。
