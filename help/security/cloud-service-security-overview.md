# AEM as a Cloud Service 安全性概觀 {#cloud-service-security-overview}

## 簡介 {#introduction}

AEM as a Cloud Service 的主動安全性方法和嚴格的程序有助於保護 AEM as a Cloud Service 解決方案和客戶機密資料的安全。持續監控數位體驗的安全性，以搶先掌握惡意活動並幫助確保客戶資料的安全。

請參考 [Adobe Experience Manager Cloud Service 安全性](https://www.adobe.com/content/dam/cc/en/security/pdfs/AEMCloudService_Security_Overview.pdf)白皮書，取得 AEM as a Cloud Service 安全性的深入概觀。
