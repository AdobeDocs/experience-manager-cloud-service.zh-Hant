# 雲加速管理器概述 {#overview-cam}

Cloud Acceleration Manager是基於雲的應用程式，旨在指導您的IT團隊從規劃到Cloud Service生活的整個過渡過程。 通過Adobe建議的最佳實踐、提示、文檔和工具，為成功遷移建立團隊，以便在遷移過程的每個階段都提供AEMCloud Service幫助。
