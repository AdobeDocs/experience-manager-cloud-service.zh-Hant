# AEMas a Cloud Service版本2022.1.0中移轉工具發行說明 {#release-notes}

此頁面概述AEM 2022.1.0中移轉工具的發行說明。

## 內容轉移工具 {#ctt-release}

### 發行日期 {#release-date-ctt}

內容轉移工具1.7.18版的發行日期為2022年1月18日。

### 新增功能 {#what-is-new-ctt}

* 切換為「內容轉移工具」中新增的提取階段，讓使用者可停用 [預復](https://experienceleague.adobe.com/docs/experience-manager-cloud-service/moving/cloud-migration/content-transfer-tool/handling-large-content-repositories.html?lang=en) 在提取期間。 為獲得最佳提取速度，小型移轉集或自上次提取後僅新增幾個blob時，應停用提取期間的預復。

### 錯誤修正 {#bug-fixes-ctt}

* 已更新預設設定，以減少提取期間的執行逾時。
