# 維護版本發行說明 {#maintenance-release-notes}

以下章節概述與 2026.3.0 功能啟用相關聯之 Experience Manager as a Cloud Service 維護版本的技術版本發行說明。

## 24678 版 {#release-24678}

以下摘要說明維護版本24678數的持續改善，該版本於2026年3月4日公開發佈。 前一個維護版本是版本 24464。

2026.3.0 功能啟用會提供此維護版本的完整功能集。如需詳細資訊，請參閱 [Experience Manager 發行藍圖](https://experienceleague.adobe.com/zh-hant/docs/experience-manager-release-information/aem-release-updates/update-releases-roadmap)。

### 增強功能 {#enhancements-24678}

* Forms-18927：在AEM Forms「檔案附件」元件中新增對自訂MIME型別和副檔名的支援，讓使用者可附加更廣泛的檔案型別。
* Forms-18211、FORMS-22936：使用者遇到協助工具問題，核取方塊在`<fieldset>`元素內未正確分組，且群組標籤未巢狀內嵌於`<legend>`，做為第一個子項。 這會影響依賴熒幕助讀程式進行導覽的殘障使用者。 以核心元件為基礎的最適化Forms現在引進了欄位集和圖例支援，以提供更好的協助工具支援。
在面板中新增「欄位集」選項，讓使用者能在表單中更有效組織和分組相關欄位。
* Forms-23880：在核心元件中新增主題編輯器支援。 此增強功能可讓使用者在核心元件中更有效率地自訂和管理主題，改善其設計彈性和工作流程。
* Forms-21772：新增Forms管理UI的版本設定支援。 此增強功能可讓使用者建立和擷取以核心元件為基礎以及以基礎元件為基礎的最適化Forms、表單片段、主題和二進位Assets的版本，進而改善資產管理和版本控制。
* Forms-23094：新增以Foundation元件為基礎的最適化Forms的使用者端剖析，讓企業客戶得以將其表單移轉至雲端。 此增強功能支援程式碼編輯器規則中的EcmaScript功能（先前不支援），有助於更順暢的移轉程式。
* Forms-23853：新增覆寫sling元件中reCAPTCHA的支援。 此增強功能可讓使用者自訂reCAPTCHA設定，提高企業客戶的彈性和安全性。
* SITES-34936：使用通用編輯器的Edge Delivery：根據發佈模型新增篩選內容片段。
* SITES-36203：使用通用編輯器的Edge Delivery：新增程式碼切換以啟用多欄位和複合多欄位支援。
* SITES-37037：使用通用編輯器的Edge Delivery：改善試算表匯入，以自動偵測分隔符號。

* SITES-40042：使用通用編輯器的Edge Delivery：將設定服務設為新網站的預設值。
