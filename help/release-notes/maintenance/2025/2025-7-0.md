# 維護版本發行說明 {#maintenance-release-notes}

## 版本 21706 {#21706}

以下摘要說明維護版本 21706 的持續改善內容，該版本已於 2025 年 7 月 24 日公開發佈。前一個維護版本為版本 21570。

>[!NOTE]
>

2025.7.0 功能啟用會提供此維護版本的完整功能集。如需詳細資訊，請參閱 [Experience Manager 發行藍圖](https://experienceleague.adobe.com/zh-hant/docs/experience-manager-release-information/aem-release-updates/update-releases-roadmap)。
