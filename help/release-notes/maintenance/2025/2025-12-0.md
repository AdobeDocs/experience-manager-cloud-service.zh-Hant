# 維護版本發行說明 {#maintenance-release-notes}

以下章節概述與 2025.12.0 功能啟用相關聯之 Experience Manager as a Cloud Service 維護版本的技術版本發行說明。

## 23482 版 {#23482}

以下摘要說明維護版本23482數的持續改善，該版本於2025年12月3日公開發佈。 前一個維護版本是 23385 版。

2025.12.0 功能啟用會提供此維護版本的完整功能集。如需詳細資訊，請參閱 [Experience Manager 發行藍圖](https://experienceleague.adobe.com/zh-hant/docs/experience-manager-release-information/aem-release-updates/update-releases-roadmap)。

### 增強功能 {#enhancements-23482}

* Assets-49770：新增惡意程式碼掃描結果的隔離通知。
* Assets-54079：套用隔離資料夾的自訂中繼資料表單。
* Assets-54083：建立排程的隔離清理機制。
* Assets-54278：從資產中移除`dam:avScanTime`屬性。
* Assets-57284：將檔案上傳限制在隔離資料夾（停用拖放）。
* Assets-57428：在Assets檢視UI中隱藏隔離資料夾。
* Assets-57626：改善非同步資產工作的重試行為。
* Assets-57879：為非同步移動/複製資產作業新增合併選項。
* Assets-58099：新增設定以停用整個環境的增強型智慧標籤。
* Assets-58136：在Search OpenAPI中實作分頁意見回饋。
* Assets-59402：為資料夾刪除API新增非同步作業端點：將套件匯出至內部區域。
* Assets-59966：將Malware Administrators群組重新命名為Quarantine Administrators。
* Assets-60166：使用VideoViewer.js而非iframe型URL。
* GRANITE-61378：許可權偵錯工具 — ListPrincipals API。
* GRANITE-63235：查詢以識別使用`cq:conf`屬性的網站，支援偵測舊頁面/版本。
* SITES-30452：包含ASO的內容API — 標題和說明建議、XWalk支援、JSON修補程式操作、IMS服務主體繫結。

### 已修正的問題 {#fixed-issues-23482}

* Assets-57430：修正Assets檢視上傳略過前置處理：匯出`repoapi.preprocessing`封裝，將RAPI更新為最新版本。
* Assets-58190：減少集合UI中不必要的高猜測總數。
* Assets-58866：修正OpenAPI回應中傳回的資產標題/說明/ID。
* Assets-58868：資產上缺少排序欄位時修正分頁。
* Assets-58920：修正大量資產匯入，略過前置處理。
* Assets-59168：修正顯示錯誤時區的掃描開始/結束時間。
* Assets-59702：資產狀態設為「無狀態」時，修正事件排序。
* Assets-59830：重新佇列非同步工作在pod終止期間停止。
* Assets-49757：惡意程式碼偵測掃描事件的修正。
* GRANITE-61019：AEM重新啟動後，第一次執行時未通知修正`gcMonitor`。
* GRANITE-62717：修正非ASCII字元的`JSafe`密碼處理。
* SITES-34331：修正非管理員使用者(`cqLiveSyncCancelled index`)的503載入轉出覆蓋逾時。

### 已知問題 {#known-issues-23482}

無。

### 安全性修正 {#security-23482}

AEM as a Cloud Service 專門負責將您的平台的安全性與效能最佳化。此維護版本解決了 4 個已確認的漏洞，強化我們提供健全系統保護的承諾。
