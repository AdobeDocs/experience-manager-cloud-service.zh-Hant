# 維護版本發行說明 {#maintenance-release-notes}

以下章節概述與 2025.5.0 功能啟用相關聯之 Experience Manager as a Cloud Service 維護版本的技術版本發行說明。

## 版本 20626 {#20626}

以下摘要說明維護版本20626數的持續改善，該版本於2025年4月29日公開發佈。 前一個維護版本是版本 20476。

2025.5.0功能啟動提供此維護版本的完整功能集。 如需詳細資訊，請參閱 [Experience Manager 發行藍圖](https://experienceleague.adobe.com/zh-hant/docs/experience-manager-release-information/aem-release-updates/update-releases-roadmap)。

### 增強功能 {#enhancements-20626}

* Assets-46413、ASSETS-46580：新增稽核狀態「預覽」。
* Assets-49542：擴充視訊與音訊轉譯與翻譯的支援語言。
* Assets-48264：擴充轉譯的PNG品質支援。

### 已修正的問題 {#fixed-issues-20626}

* Assets-50387：修正內容片段預設縮圖，以用於GenStudio。
* Assets-49006：當使用者沒有寫入許可權時，顯示視訊屬性。
* Assets-46757、ASSETS-46997：改善智慧型裁切編輯器中的協助工具。
* Assets-48018：改善Assets發佈報表中的資產參考追蹤功能。
* Assets-35846：改善製作層級與傳遞層級之間存取的一致性。
* Assets-48171：使用畫布改善Dynamic Media範本化的一致性。
* Assets-49813：改善到期通知。
* Assets-47768、ASSETS-49825、ASSETS-49008、ASSETS-48287：改善大量作業的管理與可見性。
* Assets-50003、ASSETS-50004：改善資產下載中所含轉譯的命名和控制。
* Assets-47939：改善Content Hub回應的組織方式。
* Assets-46738：改善非常大型集合的效能。
* Assets-50121：改善資產已發佈事件的可靠性。
* Assets-48490：提高影像擷取期間自動化處理的恢復能力。
* Assets-28106、ASSETS-49404：提高全文搜尋的穩健性。
* Assets-50006、ASSETS-50423：改善大型資料夾的搜尋和周遊效能。
* Assets-46021：改善Safari和行動瀏覽器的視訊顯示。
* Assets-49002：改善編輯Dynamic Media範本的處理方式。
* Assets-48376： Content Hub UI中的其他改善。
* Assets-48504、ASSETS-49378： UI行為的其他改善。
* Assets-49540：將資產關係OpenAPI移出實驗階段。
* Assets-40284：更新Adobe Stock整合的相關檔案。
* Assets-49739：致力於整合資產選擇器中的Figma。

#### AEM 指南 {#guides}

* GUIDES-21734：如果透過代碼片段新增元素，或透過範本建立元素，則即使在XMLEditorConfig中啟用了自動產生ID選項，仍無法為元素產生新ID。
* GUIDES-25969：如果DITA主題中的外部連結缺少`scope=external`屬性，HTML5發佈會失敗，而不會在錯誤記錄中指出缺少此屬性的檔案，尤其是啟用微服務時。
* GUIDES-27288：無法傳遞中繼資料屬性以對應使用新AEM Sites發佈產生的登入頁面。

如需更多有關該版本中新增功能和增強功能以及已修復問題的資訊，請查看 [Experience Manager Guides 發行藍圖](https://experienceleague.adobe.com/zh-hant/docs/experience-manager-guides/using/release-info/aem-guides-releases-roadmap)。

AEM as a Cloud Service 專門負責將您的平台的安全性與效能最佳化。此維護版本解決了 11 個已確認的漏洞，強化我們提供健全系統保護的承諾。
