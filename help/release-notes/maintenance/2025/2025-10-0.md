# 維護版本發行說明 {#maintenance-release-notes}

以下章節概述與 2025.10.0 功能啟用相關聯之 Experience Manager as a Cloud Service 維護版本的技術版本發行說明。

## 22758 版 {#22758}

以下所彙整的是 22758 版維護版本持續改善的內容，該版本於 2025 年 10 月 1 日公開發佈。前一個維護版本是 22450 版。

2025.10.0 功能啟用會提供此維護版本的完整功能集。如需詳細資訊，請參閱 [Experience Manager 發行藍圖](https://experienceleague.adobe.com/zh-hant/docs/experience-manager-release-information/aem-release-updates/update-releases-roadmap)。

### 增強功能 {#enhancements-22758}

* ASSETS-56227：重新命名 adobe-countdown-timer 修飾詞。
* CNTBF-493：將內容回流套組版本提升至 2.0.28。
* CQ-4361110：Granite 翻譯。
* CQ-4361112：最新的 AEM 翻譯。
* GRANITE-56026：改善權限 API 狀態代碼回應。
* GRANITE-61015：將 `org.apache.commons.io.channels` 封裝新增至公開匯出的清單。
* GRANITE-61167：Felix 記錄更新至最新的 OSGI 規格。
* GRANITE-61167：更新多個 Apache Felix 相依性。
* GRANITE-61169：改善受保護字串的檢查。
* GRANITE-61622：更新多個 Apache Sling 相依性。
* GRANITE-61663：將 `com.adobe.granite.repository.indexdefs-1.0.2` 新增至快速入門。
* GRANITE-61811：將 `com.adobe.granite.repository-2.0.0` 新增至快速入門。
* SITES-32014：監聽外部事件，更新服務註冊。
* SITES-34277：修正頁面翻譯工作流程中的封鎖錯誤。
* SKYOPS-108706：將發行版本切換套組更新至最新版本 (etag 快取)。
* SKYOPS-114210：更新至最新版本的 aem.pss.service 套組。
* SKYOPS-116171：更新至 Sling ResourceResolver 1.12.12。
* SKYOPS-119811：發行 dispatcher-publish 2.0.258。

### 已修正的問題 {#fixed-issues-22758}

* GRANITE-61875：修正「無效運算式評估」的觸發程序：作者無法儲存內容片段且資產無法下載。
* SITES-22059：修正 PDF 檢視器元件中的 JS 錯誤。核心元件網站 > PDF 檢視器中的「檔案預覽無法使用」字串未本地化。
* GRANITE-59704：修正 htmllibmanager.debug 導致編輯模式失敗的狀況。
* GRANITE-61042：將 FELIX-6796 (ServiceTracker NPE 修正) 整合至 AEM Felix 網頁控制台套組中。
* GRANITE-61165：Workspace.copy() 擲回 RepositoryException。
* GRANITE-61875：將 ui.commons 更新至 5.10.50。

### 已知問題 {#known-issues-22758}

無。

### 已過時的功能和 API {#deprecated-22758}

[「已過時和已移除的功能及 API」](/help/release-notes/deprecated-removed-features.md)文件中詳細介紹 AEM as a Cloud Service 中已過時和已移除的功能及 API。

### 安全性修正 {#security-22758}

AEM as a Cloud Service 專門負責將您的平台的安全性與效能最佳化。此維護版本解決 13 個已確認的漏洞，強化我們提供健全系統保護的承諾。
