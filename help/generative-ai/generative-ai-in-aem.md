# AEM as a Cloud Service中的Generative AI {#generative-ai-in-aem-as-a-cloud-service}

適用於Adobe Experience Manager (AEM) as a Cloud Service的Adobe Generative AI可讓品牌透過編輯介面建立具備複製和影像產生功能的高效能體驗。 使用創作AI，您可以自主建立以下內容：

* 品牌感知 — 建立結合品牌語調、風格准則和其他特定要求的內容。
* 依對象個人化 — 運用精細的內容效能深入分析，為跨所有管道的任何角色量身打造內容。
* 地區特定 — 超越翻譯，根據地區偏好和行為模式調整內容。
* 持續最佳化 — 執行實驗以判斷哪些變數表現最佳，並使用見解來改善未來AI產生的內容。

Generative AI在整個AEM中使用：

* [產生變化版本：與 AEM 編輯器整合](/help/generative-ai/generate-variations-integrated-editor.md)

  產生變化版本使用生成式人工智慧 (AI)，根據您的輸入建立內容變化版本。建立變化後，您可以在網站上使用內容，也可以使用Edge Delivery Services的Experimentation功能來衡量其成功與否。

   * 和舊版[產生變數](/help/generative-ai/generate-variations.md)

     >[!NOTE]
     >
     >建議您存取整合至AEM編輯器的[產生變數](/help/generative-ai/generate-variations-integrated-editor.md)，因為先前版本未來將停用。

   * [產生變化版本發行說明](/help/generative-ai/release-notes-generate-variations.md)

* [Content Hub 概觀](/help/assets/product-overview.md)

  Content Hub 可讓您使用 Adobe Express 來建立新內容 (如果您具有 Adobe Express 權益)。您可以使用易於使用的工具來編輯現有內容、使用範本和品牌元素製作品牌變化版本，並使用 Adobe Firefly 的最新 GenAI 功能來建立新內容。
