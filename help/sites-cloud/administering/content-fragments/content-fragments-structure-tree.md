# 內容片段樹狀結構 {#content-fragment-structure-tree}

<!--
hide: yes
index: no
hidefromtoc: yes
-->

使用AEM中內容片段編輯器的結構樹功能來協助您瞭解您的Headless內容。

在內容片段編輯器中，您可以選取「結構樹」圖示：

![內容片段樹狀結構](assets/cfm-structuretree-01.png)

這將在左側窗格中開啟片段結構的表示法。 您可以使用此項來導覽所參考的片段，以及瀏覽至所參考的片段。 選取一個參考會開啟該片段供您編輯。

>[!NOTE]
>
>使用主面板中的階層連結，您可以導覽回您的起點。

![內容片段樹狀結構](assets/cfm-structuretree-02.png)
