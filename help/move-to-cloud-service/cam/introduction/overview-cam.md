# Cloud Acceleration Manager概述{#overview-cam}

Cloud Acceleration Manager為客戶提供一站式的移轉體驗，提供初始評估服務，將客戶指向正確的工具、檔案和最佳實務，以確保順利以AEM as aCloud Service上線。
