# 資產工作流程移轉工具 {#asset-workflow-migration}

資產工作流程移轉工具可用來自動將資產處理工作流程從AEM的內部部署或AMS部署移轉至處理設定檔和OSGi設定，以便用於AEM Assets的雲端服務。

## 簡介 {#introduction}

本節介紹有關資產工作流遷移工具的資源和實施詳細資訊。

此公用程式可讓AEM開發人員將現有的AEM資產處理工作流程移轉至AEM做為雲端服務。

## 安裝資產工作流程移轉工具 {#installing-tool}

請參閱 **[Git資源： AEM Assets as a Cloud Service - Workflow Migration](https://github.com/adobe/aem-cloud-migration)**，以瞭解如何從來源安裝和建立程式碼。
