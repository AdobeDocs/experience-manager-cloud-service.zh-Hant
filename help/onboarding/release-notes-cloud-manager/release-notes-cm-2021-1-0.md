* 資產生產實例有時可能會將&#x200B;**環境**&#x200B;詳細資訊頁面上的品牌入口網站狀態顯示為&#x200B;*待定*，而不允許使用者採取任何動作。

* 已解決在環境建立或刪除中遇到的罕見故障。
