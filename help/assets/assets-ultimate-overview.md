Assets as a Cloud Service Ultimate提供進階DAM功能。 AEM Assets Ultimate的設計用途是管理複雜的內容供應鏈，確保每一項內容在所有管道皆能正常運作。

## 為何選擇Assets Ultimate？ {#why-ultimate-existing-new-users}

Assets as a Cloud Service Ultimate提供多種主要優點，協助您有效管理組織的資產需求，例如：

* 透過與這些使用者型別相關的更多使用者型別和許可權（例如Collaborator使用者、超級使用者和受限使用者）提供更大的彈性。

* 透過Content Hub無縫分發資產。

* 使用Adobe Express與Firefly的AI支援內容建立和混合功能。

* 為全新和現有使用者提供更流暢的入門或升級體驗。

## Assets Ultimate的重要功能 {#capabilities-assets-ultimate}

Assets as a Cloud Service Ultimate可讓您執行各種重要的數位資產管理作業，例如：

* **資產管理和程式庫服務**&#x200B;：可讓使用者在集中式存放庫中擷取、儲存、編目、控制、管理及控管品牌數位資產的工具

* **搜尋、探索和Collaboration**：可讓使用者瀏覽、探索、共用及共同使用所需資產的工具，以建立豐富的客戶體驗。

## 可用的使用者型別和許可權有哪些？ {#available-user-types}

Assetsas a Cloud Service提供四種使用者型別。 每種使用者型別提供一組不同的許可權。 使用者型別包括：

* **管理員**：配置組織內其他三種使用者型別的標準管理員使用者。

* **受限使用者**：受限使用者可以使用AEM Assets Content Hub入口網站存取及運用貴組織核准的資產。

* **Collaborator使用者**：身為Collaborator使用者，您可以：

   * 透過貴組織在其他Adobe產品和非Adobe應用程式中可用的Assets整合，使用來自Experience Manager的資產。

   * 運用專業設計的範本、品牌套件、Adobe Stock資產等，以內建Adobe Express和Firefly建立及編輯資產。

   * 使用AEM Assets Content Hub入口網站存取及運用貴組織核准的資產。

* **超級使用者**：身為超級使用者，您可以：

   * 存取所有AEM Assets功能，包括管理資產、中繼資料，以及有關數位資產的整體治理和自動化。

   * Work with assets from Experience manager via integrations of Assets available to your organization in other Adobe and non-Adobe applications.

   * Create and edit assets using built-in Adobe Express and Firefly leveraging professionally designed templates, brand kits, Adobe Stock assets, and so on.

   * Access and leverage approved assets from your organization using AEM Assets Content Hub portal.
