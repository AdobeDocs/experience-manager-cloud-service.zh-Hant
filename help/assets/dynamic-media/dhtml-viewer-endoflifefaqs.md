# DHTML檢視器生命週期結束 {#dhtml-viewer-end-of-life}

自2014年1月31日起，DHTML檢視器平台正式終止服務。

如需最新支援的檢視器，請參閱[Dynamic Media檢視器參考指南](https://experienceleague.adobe.com/docs/dynamic-media-developer-resources.html?lang=zh-Hant)。
