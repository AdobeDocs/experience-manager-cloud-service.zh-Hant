# DHTML檢視器生命週期結束 {#dhtml-viewer-end-of-life}

DHTML檢視器平台已正式終止服務，於2014年1月31日起生效。

請參閱 [Dynamic Media檢視器參考指南](https://experienceleague.adobe.com/docs/dynamic-media-developer-resources.html) 以取得最新支援的檢視器。
