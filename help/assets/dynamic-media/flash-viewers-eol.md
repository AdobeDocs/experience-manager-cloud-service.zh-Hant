# Flash檢視器生命週期結束{#flash-viewers-end-of-life-notice}

自2017年1月31日起，對Flash檢視器平台的支援即將正式終止。

如需最新支援的檢視器，請參閱[動態媒體檢視器參考指南](https://experienceleague.adobe.com/docs/dynamic-media-developer-resources/library/home.html)。
