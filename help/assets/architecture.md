# 資產架構：[!DNL Cloud Service]解決方案{#assets-architecture}

[!DNL Experience Manager Assets] 與最 [!DNL Cloud Service] 新產品6. [!DNL Adobe Experience Manager] 5的基礎和比較 [!DNL Experience Manager] 良好。本產 [!DNL Cloud Service] 品具有調整後的架構和功能，將成為雲端原生服務。基礎架構的變更不允許所有Experience Manager 6.5功能都屬於[!DNL Cloud Service]。

要查看資產微服務的體系結構概述，請參閱[資產微服務概述](asset-microservices-overview.md#asset-microservices-architecture)。
