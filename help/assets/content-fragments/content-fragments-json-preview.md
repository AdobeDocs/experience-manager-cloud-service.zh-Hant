# 預覽- JSON表示法{#preview-json-representation}

當開發內容片段的模型做為無頭實作的一AEM部分時，您可能會想要檢視內容片段的範例JSON輸出，如同根據模型。 例如，若要瞭解最終輸出的外觀。 當驗證模型JSON結構時，這可能很有幫助，可能會針對每個資料類型使用預設範例內容。
