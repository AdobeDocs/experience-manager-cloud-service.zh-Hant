# 使用AEM MCP設定游標 {#setup-cursor}

請依照下列步驟，將Cursor連線至AEM的MCP伺服器。

* 在Cursor的MCP設定中，建立一個新的MCP伺服器專案，包含一個或多個AEM MCP URL。
* 出現提示時，使用您的Adobe ID進行驗證。
* 或者，按一下工具名稱來啟用或停用個別工具。 預設會啟用所有工具。
* 使用游標的編輯器或聊天，叫用AEM工具作為開發或內容工作流程的一部分。

>[!NOTE]
>
>Cursor使用者介面可能會有所變更，而且不是決定性的。 這些指示僅供說明用途。

1. 開啟&#x200B;**資料指標設定**，以便您可以設定資料指標如何連線到MCP伺服器。

   ![游標設定對話方塊。](assets/cursor-1.png)

1. 開啟&#x200B;**工具和MCP**，然後選擇&#x200B;**新增自訂MCP**&#x200B;以啟動自訂MCP伺服器專案。

   ![包含新增自訂MCP伺服器的選項的[工具和MCP]面板。](assets/cursor-2.png)

1. 在自訂MCP伺服器表單上，輸入&#x200B;**名稱**、您的AEM MCP **URL** （或URL）和任何其他必要欄位，然後&#x200B;**儲存**。

   ![資料指標中的自訂MCP伺服器設定表單。](assets/cursor-3.png)

1. 當連線對話方塊出現時，按&#x200B;**連線**&#x200B;完成登入，以授權新的MCP伺服器。

   ![游標中新MCP伺服器的連線對話方塊。](assets/cursor-4.png)

1. 在&#x200B;**聊天**&#x200B;或編輯器中，撰寫呼叫&#x200B;**AEM工具**&#x200B;的提示，讓設定的MCP伺服器參與您的工作流程。

   ![提示游標使用新的AEM MCP服務。](assets/cursor-5.png)
