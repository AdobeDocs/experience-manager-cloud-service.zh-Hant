# AEM代理程式概觀 {#overview-of-agents-in-aem}

Adobe Experience Manager (AEM) as a Cloud Service提供這些代理程式，可加速內容建立並自動協調變更。

>[!IMPORTANT]
>
>這些代理程式僅適用於AEM as a Cloud Service和Edge Delivery Services。
>
>* 目前無法用於AEM 6.5，也無法用於AEM 6.5 LTS。
>* 內部部署或Managed Services均無法提供AEM。

>[!IMPORTANT]
>
>若要使用代理程式，您必須先註冊Explorer程式。 您可以在[此處，找到有關AEM Agents總管程式的詳細資訊。](/help/release-notes/release-notes-cloud/release-notes-current.md#aem-beta-programs)

## Brand Experience Agent {#brand-experience-agent}

[品牌Experience Agent](/help/ai-in-aem/agents/brand-experience/overview.md)可自動執行大量及大量工作。

* **內容更新** — 更新頁面或內容片段
* **表單建立** — 建立表單或匯入表單
* **開發疑難排解** — 讓技術角色（開發人員和管理員）能夠簡化程式碼的建立、偵錯、部署和最佳化。

Brand Experience Agent可讓團隊將長達數週的手動程式轉換為快速的AI輔助工作流程，讓每個體驗保持最新且一致，以協助企業達成目標。

## 內容建議程式代理程式 {#content-advisor-agent}

[Content Advisor代理程式](/help/ai-in-aem/agents/content-advisor/overview.md)可透過套用自然語言指令來建立適合頻道的變體，簡化下列工作，以轉換使用者探索、調整及調整資產的方式。

* **探索** — 提供AEM內容，提供精簡的免費探索體驗。 智慧型搜尋Assets、內容片段和最適化Forms，提供相關素材，例如影像、影片、PDF檔案、文章和表單範本。
* **內容最佳化** — 透過套用自然語言指示來建立適合管道的變數，轉變使用者如何調整並調整資產。 解譯使用者意圖並自動執行複雜的編輯任務以執行任務，包括產生新轉譯、調整視覺屬性、變更背景或為特定數位頻道準備資產。

代理程式會解譯使用者意圖並自動執行複雜的編輯任務，包括產生新轉譯、調整視覺屬性、變更背景或為特定數位頻道準備資產。

## Experience Modernization Agent {#site-modernization}

[Experience現代化代理程式](/help/ai-in-aem/agents/modernization/overview.md)透過讓網站移轉和持續演化的工作快速順暢地進行，釋放了Edge Delivery Services的全部價值(包括AEM製作)。

## 治理代理程式 {#governance-agent}

[治理代理程式](/help/ai-in-aem/agents/governance/overview.md)可保障AEM的品牌完整性和合規性。 透過強制實行安全性、法規及品牌政策，代理程式會執行必要的工作，以確保每次互動與啟用都遵循既定的標準。

## 限制 {#limitations}

請留意AEM代理程式的下列一般限制：

* 這些代理程式僅適用於AEM as a Cloud Service和Edge Delivery Services。
* 目前無法用於AEM 6.5，也無法用於AEM 6.5 LTS。
* 目前無法在AEM內部部署或Managed Services中使用。

如需代理程式的特定限制，請參閱個別頁面。
