# AEM代理程式概觀 {#overview-of-agents-in-aem}

Adobe Experience Manager (AEM) as a Cloud Service提供這些代理程式，可加速內容建立並自動協調變更。

>[!IMPORTANT]
>
>這些代理程式僅適用於AEM as a Cloud Service和Edge Delivery Services。
>
>* 目前無法用於AEM 6.5，也無法用於AEM 6.5 LTS。
>* 內部部署或Managed Services均無法提供AEM。

>[!IMPORTANT]
>
>若要使用代理程式，您必須註冊。

## Experience Production Agent {#experience-production-agent}

[Experience Production Agent](/help/ai-in-aem/agents/production/overview.md)可自動執行大量工作及大量工作。 強化團隊，將長達數週的手動流程轉換為快速的AI輔助工作流程，讓每個體驗保持最新且一致，以協助企業達成目標。

## 內容最佳化代理程式 {#content-optimization-agent}

[內容最佳化代理程式](/help/ai-in-aem/agents/content-optimization/using.md)透過套用自然語言指示來建立適合管道的變數，以轉換使用者如何調整並調整資產。 代理程式會解譯使用者意圖並自動執行複雜的編輯工作，以執行包括產生新轉譯、調整視覺屬性、變更背景或為特定數位頻道準備資產等工作。

## 探索代理程式 {#discovery-agent}

[探索代理程式](/help/ai-in-aem/agents/discovery/using.md)提供AEM內容，提供精簡的免費探索體驗。 它會在Assets、內容片段和最適化Forms中聰明地搜尋，以傳遞相關素材，例如影像、影片、PDF檔案、文章和表單範本。

## 開發代理程式 {#development-agent}

[開發代理程式](/help/ai-in-aem/agents/development/overview.md)透過簡化程式碼的建立、偵錯、部署和最佳化，來強化技術角色（開發人員和管理員）。

## 治理代理程式 {#governance-agent}

[治理代理程式](/help/ai-in-aem/agents/governance/overview.md)可保障AEM的品牌完整性和合規性。 透過強制實行安全性、法規及品牌政策，代理程式可確保每次互動與啟用都遵循既定的標準。

## 限制 {#limitations}

目前存在的限制包括：

* 這些代理程式僅適用於AEM as a Cloud Service和Edge Delivery Services。
   * 目前無法用於AEM 6.5，也無法用於AEM 6.5 LTS。
   * 目前無法在AEM內部部署或Managed Services中使用。

如需代理程式的特定限制，請參閱個別頁面。
